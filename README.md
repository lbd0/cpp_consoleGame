### 💡주제   
떨어지는 똥을 피해 오래 살아 남는 것이 목표인 슈팅 게임.
> - 떨어지는 똥을 피해 오래 살아남아야 함.
> - 3번 똥을 맞으면 GameOver.
> - 살아남은 시간 측정.
> - 가장 오래 살아남은 시간 저장됨. (최고 기록)
